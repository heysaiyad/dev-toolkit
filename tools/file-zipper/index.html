How to Use

  • Upload Files: Drag and drop files or click to browse. You can upload any file type.
  • Reorder Files: Drag the handle icon to rearrange files in your desired order.
  • Remove Files: Click the trash icon to remove individual files from the list.
  • Create ZIP: Click the "Create ZIP" button to compress all files into a downloadable ZIP archive.
  • Privacy First: All processing happens in your browser. No files are uploaded to any server.
