About Hash Functions

MD5

Message Digest Algorithm 5 produces a 128-bit hash (32 hexadecimal characters). Fast but considered cryptographically broken for security purposes.

SHA-256

Secure Hash Algorithm 256-bit produces a 256-bit hash (64 hexadecimal characters). Cryptographically secure and widely used.

Important: Hash functions are one-way operations. You cannot reverse a hash to get the original text.
